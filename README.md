# Supermarket Sales Dashboard

An interactive web dashboard built with **Streamlit** to explore and visualize supermarket sales data through dynamic filters and KPIs.

**[Live App](https://craftedbygaby-supermarket.streamlit.app/)**

---

## Project Context
**Program:** IronHack Data Analytics Bootcamp — Week 6  
**Objective:** Build an interactive data dashboard using Streamlit, applying filters and visualizations to explore supermarket sales patterns.

---

## Features
- Filter data by **city**, **gender**, **minimum rating**, and **payment method**
- **KPI cards** showing total sales, average rating, total transactions, and average transaction value
- **Sales over time** line chart
- **Top 5 transactions** table
- Toggle to show/hide the raw dataset

---

## Tools & Skills
- **Python** (Pandas, Matplotlib)
- **Streamlit** for interactive dashboard building
- Data filtering, aggregation, and visualization
- `@st.cache_data` for performance optimization

---

## How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run streamlit_supermarket_app.py
```

---

## Dataset
Supermarket sales data sourced from the IronHack Data Bootcamp repository, loaded directly from GitHub.
